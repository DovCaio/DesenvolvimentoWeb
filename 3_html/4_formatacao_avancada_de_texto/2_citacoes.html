Citações em bloco

Se uma seção de conteúdo em nível de bloco for citada em algum outro lugar, você deverá envolvé-la em um elemnto blockquote para indicar isso e incluir uma url

The HTML <blockquote> Element (or HTML Block Quotation Element) indicates that the enclosed text is an extended quotation.

Citações em linhas

Funcinam da mesma maneira, exceto que vamos marcar somente uma seção de algum conteudo de uma tag ao invés de embuirmos toda ela, são usadas para citações curtas que não necessitam de quebras de parágrafos.

The quote element — <q> — is intended for short quotations that don't require paragraph breaks.
