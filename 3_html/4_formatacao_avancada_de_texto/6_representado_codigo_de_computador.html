Tag code
Serve para marcar partes genéricas do código
Tag pre
Para reter espaço em branco, quando colcamos muitos espaços em branco o navegador so rendeiza apenas o primeiro espaço, quando escrevemos código precisamos organizar ele, portanto, quando vamos usar os espaços para organizar o código usamos essa tag para que todos os espaços sejam renderizados.

kbd
Marcar a entrada do teclado (e outros tipos) inserida no computador
samp
Marcar a saida de um programa de computador.
