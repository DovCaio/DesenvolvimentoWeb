Conteúdo de vídeo e áudio

Elemento video

Permite encorporar um vídeo com muita facilidade.

Da mesma maneira que temos o src em imagens temos em video e funciona da mesma maneira, ainda temos um atributo chamado control que controla o estado do vídeo, como pausa, play entre outras, e com isso podemos criar nossa própria api

Usando vários formatos para melhorar a compatibilidade

As vezes o nevegador não tem compatibilidade com determinado tipo de vídeo, seja por ser uma versão antiga ou por outro motivo, temos algumas problemas em escolher qual tipo de arquivo, uma vez que temos muitas coisas para levar em consideração, e com essas coisas a se considerar o melhor é ter dois tipos de vídeo: mp4 e webm. Tendo esses dois garantimos que o vídeo vai ter compatibilidade com a grande maioria dos navegadores, além de ter grande compatibiliadade com navegadores de celulares.

É importante usarmos o atributo type, por que assim o navegador sabera o MIME type que redenderizam cada tipo de arquivo, e com o atributo type atribuido o valor certo o navegador sabera exatamente qual usar assim não perdendo tempo procurando qual.

Outros atributos

Intuitivamente, o width é a largura, o height é a largura, o autoplay faz o vídeo ser iniciado quando a página é carregada, o loop significa que o vídeo vai recomeçar quando terminar, o muted é para o vídeo iniciar mutado e o poster é a tumbnail do vídeo.
