Inserindo imagens no HTML

para tal basta usarmos a tag img, e dentro do atributo src colocarmos o diretório, seja ele relativo ou absoluto

Aviso: A maioria das imagens tem direitos autorais. Não exiba uma imagem em sua página da web, a menos que: 1) você é o dono da imagem 2) você recebeu permissão explícita e por escrito do proprietário da imagem, ou 3) você tem ampla prova de que a imagem é, de fato, de domínio público.

Violações de direitos autorais são ilegais e antiéticas. Além disso, nunca aponte seu atributo src para uma imagem hospedada no site de outra pessoa à qual você não tem permissão para vincular. Isso é chamado de "hotlinking". Mais uma vez, roubar a largura de banda de alguém é ilegal. Ele também torna a página mais lenta, deixando você sem controle sobre se a imagem é removida ou substituída por algo embaraçoso.

Texto alternativo

É um atributo de img, chamado de alt, ele contem o texto que vai aparecer caso a imagen não seja rederizadar por algum motivo.

Motivos esses que podem ser:

  • O usuário é deficiente visual e usa um leitor de tela para ler a web para eles. De fato, ter o texto alternativo disponível para descrever imagens é útil para a maioria dos usuários.
  • Como descrito acima, você pode ter digitado o nome do arquivo ou caminho errado.
  • O navegador não suporta o tipo de imagem. Algumas pessoas ainda usam navegadores somente de texto, como Lynx, que alternativamente exibe o texto alternativo das imagens.
  • Você pode fornecer texto para os mecanismos de pesquisa utilizarem. Por exemplo, os mecanismos de pesquisa podem corresponder o texto alternativo às consultas de pesquisa.
  • Os usuários desativaram as imagens para reduzir o volume e as distrações de transferência de dados. Isso é especialmente comum em telefones celulares e em países onde a largura de banda é limitada e cara.

O que realmente devemos escrever dentro do alt? Depende do por que a imagen está lá em primeiro lugar. O que perde se sua imagen não aparecer:

  • Decoração. Se a imagem é apenas decoração adicionar um espaço em branco no alt. alt="".
  • Conteúdo. Caso a imagen tenha um sentido no conteúdo e o que tem nela não está bem explicado no conteúdo principal, escrevemos um breve texto no alt, do contrário deixamos em branco.
  • Links. Podemos transforma uma imagem em um link caso venhamos a colcar ela dentro de uma tag a, nesse caso podemos escrever o link tanto no alt da tag img, quanto no href da tag a.
  • Texto. Devemos buscar ao máximo não colocar textos em imagens, porém caso não exista outra forma, fornecemos o mesmo texto no alt.

Largura e altura

Podemos usar os atributos width e height, para largura e altura respectivamente, isso é interessante, uma vez que, enquanto a imagem não é carregada, caso essess atributos tenham valores, o navegador define uma area com essas dimensões.

No entanto devemos colocar as mesmas dimensões originais da imagem, caso contrário ela ficará granulada, ou muito pequenas, despediçando larguara de banda, por isso devemos usar um editor de imagens antes.

Titulos

Img também tem uma atributo title, que nos dar mais informações de suporte.

Anotar imagens com figuras e legendas de figuras

Podemos querer criar uma legenda em uma imagem e para isso podemos criar uma div e dentro dela cooloca uma imagem e legendas, entretanto isso não funciona para que esta acessando por mecanismos de leitura. Para tal usamos a tag figure, a qual podemo fazer algom como foi apresentado, porém dentro dela podemos usar a tag figcaption, que serve justamente de legenda.

Nesse caso a alt não deve conter o mesmo conteúdo de figcaption, por que quando a imagem não é carregada os dois vão aparecer.

Imagens de fundo CSS

Podemo incorporar imagens em páginas web, inserido isso no css, por exemplo:


    

        p {
            background-image: url("images/dinosaur.jpg");
        }
