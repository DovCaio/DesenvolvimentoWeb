Elementos de layout não semânticos

Temos situações que não conseguimos encontra um elemento semântico para determinado conteúdo ou elementos. Quando isso acontece é interessante agrupar esses elementos para afetá-los com uma única entidade com algum CSS ou JS. Para casos como esse HTML oferece as tags span, que é inline e div que é inblock, devemos usa-los sempre atribuindo uma classe a eles atribuindo algsum tipo de rótulo.
