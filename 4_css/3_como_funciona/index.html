Como o html é renderizado?

De uma forma simplificada.

  1. O navegador carrega o
  2. Ele converte o html para um DOM. O DOM representa o documento na memória do computador.
  3. O navegador requisita a maioria dos recursos que estão lincados no documento, como imagens, vídeos e audios
  4. O navegador analiza o css encotrado e interpreta as diferentes regras.
  5. A árvore de renderização é organizada na estrutura e deve aparecer depois das regras de estilo serem aplicadas ao documento.
  6. O visual de visualização da página é mostrado na tela.

Sobre o DOM

O DOM é uma estrutura árborea (three-like). Cada elemento, atributo, ou fragmento de texto na linguagem de marcação torna-se um DOM node na estrutura de arvore. Os nodes são definidos por meio do relacionamento com outros nodes presentes. Alguns elementos são pais e outros filhos.
