Estilizando Links

Estados de links

Link
Um link que tem destino, estilizamos com a pseudo-classe :link
Visited
Um link que já foi visitado (existe no histórico do navegador), estilizando a pseudo-classe :visited
Houver
Um link que o mause está em cima, estilizado usando a pseudo-classe :houver
Focus
Um link que está focado (exemplo: movido pela tecla tab ou algo similar, ou programicamente focados usando HTMLElement.focus()), isso é estilizado com a pseudo-classe :focus
Active
Um link que foi clicado, usando a pseudo-classe :active

Mundando caracteristica padrões dos links

Podemos usar o próprio colors para mudar a cor da fonte, cursor para mudar a aparencia do mouse quando ele está em cima do link e outline, se parece com o border
