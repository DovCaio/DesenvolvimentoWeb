Como Css é estruturado

Folha de estilo externa

E quando usamos o link dentro do elemento head, e dentro do href colocamos o diretório do Css esse css é o css externo.

Folha de estilo interna

E quando usamos o elemento style dentro do elemento head, assim colocando todo conteúdo Css dentro do mesmo arquivo do html.

Isso pode ser útil em algumas circunstâncias

Inline

É quando usamos o css diretamente no elemento

Seletores

Um seletor é o modo o qual nós apontamos para alguma coisa.

Cada regra CSS começa com um seletor ou uma lista de seletores para informar ao navegador em qual elemento ou elementos as regras devem ser aplicadas.

Exemplos:

        
h1
a:link
.manythings
#onething
*
.box p
.box p:first-child
h1, h2, .intro

Especificidade

A momentos em que dois seletores selecionam o mesmo elemento. 

        
            .special {
                color: red;
              }
              
              p {
                color: blue;
              }

Nesse caso quem vai vencer e acabar sendo renderizada é a da classe, porém caso tenhamos dois seletores que tem a mesma priridade, como dois seletores p, o que aparecer por último é o que vai se renderizado.

Funções

Temos como atribuir um valor a uma propriedade css usando calc, ex: calc(90% - 30px).

A propriedade trasform pode receber a função rotate()

Temos isso também em background-image, color e por ai vai.

@rules

Serve para dizes ao CSS como se comportar, por exemplo, podemos importar outro css:

@import "style2.css"

@media nos permite aplicar css somente se determinada condição for atendida, como por exemplo a resolução da tela ser de um determinado tamanho. 

        
            body{

                background-color: pink;

            }

            @media (min-width: 30em){

                body{

                    background-color: blue; 
                
                }

            }

Atalhos

Algumas propriedades são abrevições para outras: font, background, padding, border e margin. Elas nos permitem definir vários valores de propriedades em uma única fila. 

        
            h1{

                padding: 10px 15px 15px 15px;

            }

            /*Faz a mesma coisa de:*/

            h1{

                padding-top: 10px;
                padding-right: 15px;
                padding-bottom: 15px;
                padding-left: 15px;

            }
