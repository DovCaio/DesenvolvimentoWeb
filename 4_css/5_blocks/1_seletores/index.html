Seletores

Seletores são usados para direcionar elementos HTML em nossas páginas da web que queremos ultilizar

  • O que é?
  • Listas de Seleção
  • Tipos de seletores

O que é um seletor?

É a primeira parte de uma regra CSS.

            
    
                h1 {
    
    
                    color: blue;
    
                }

O seletor nesse caso é o h1

Listas de seleção

Podemos combinar vários seletores para definirmos uma só regra

    
            
                h1, .special{
    
                    color: blue;
                }

Assim todas os titulos h1 e os elementos que tem a classe .special vão ser afetados por esssa regra

Se um seletor estiver errado todas sua regra será ignorada, por isso caso tenhamos uma lista de seletores e um deles estiver errado a regra vai ser ignorada para os outros seletores, mesmo se eles estiverem corretos.

Tipos de Seletores

Seletores de tipo, classe e id

Tem como alvo um elemento HTML, como h1:

            
                h1{
    
                }

Seletores que direcionam uma classe:

            
                .box{
    
                }

Seletores que direcionam um id:

            
                #unique{
    
                }

Seletores de atributos

Selecionam elementos com base na presença de determinado atributo.

            
                a[title]{
    
                }

Ou até mesmo faça uma seleção com base na presença de um atributo com valor específico:

            
                a[href="https://exemplo.com"]{
    
                }

Pseudo classes e Pseudos-elementos

Seleciona um elemento a depender do seu estado atual.

            
                a:hover{
    
                }

Também inclui pseudo-elementos que selecionam uma determinada parte de uma elemento em vez do próprio elemento. 

            
                p::first-line{ /*Seleciona a primeira linha de um paragrafo*/
    
                }

Combinadores

            
                article > p{ /*Seleciona paragrafos que são filhos de article's*/
    
                }

Seletores de tipo

As vezes é chamado de seletor de nome de tag ou seletor de elemento

            
                span {
                    background-color: yellow;
                  }
                  
                  strong {
                    color: rebeccapurple;
                  }
                  
                  em {
                    color: rebeccapurple;
                  }

Seletor universal

Indicado por um astrisco (*). Carrega tudo dentro do documento.

Exemplo: Vamos retirara todas as margens de todos os elementos. 

            
                *{
                    margin: 0;
                }

Seletor de classe

Atribuimos um valor ao atributo class do elemento e assim podemos selecionar atraves dessa notação: .nome-da-classe

Segmentação de classes em elementos específicos

Podemos criar seletores que tem como alvo elementos específicos com a classe aplicada

            
                span.nome-da-classe{

                    background-color: yellow;

                }

Segmentar um elemento que tenha mais de uma classe aplicada 

            
                nome-da-primeira-classe.nome-da-segunda-classe{

                    color: blue;

                }

Obs: Para darmos diversas classes aos elementos basta separar cada uma por espaço

Seletores de id

Começa com o # em vez de um ponto, mas é usada da mesma forma que o seletor de classe. No entanto, um id so deve ser usado uma vez por página e os elementos só podem ter um id aplicados a eles 

            
                #nome-id{

                    color: red;

                }
