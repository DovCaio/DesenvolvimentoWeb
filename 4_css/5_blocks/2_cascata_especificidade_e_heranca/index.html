Cascata, Especificidade e Herança

Cascata

Quando duas regras da mesma camada de cascata se aplicam e ambas têm igual especificidade, aquela que foi definida por último é a que será utilizada 

            
                h1{
                    color: red;
                }
                h1{
                    color: blue;
                }

O h1 vai assumir o color blue no exemplo acima.

Especifidade

Se vários blocos de estilo tiverem seletores diferentes que configuram a mesma propriedade com valores diferentes e visam o mesmo elemento, especificidade decide o valor da propriedade que será aplicada ao elemento.

  • Um seletor de elemento tem a menor especificidade, os seletores de pseudoelementos tem a mesma especificidade.
  • Um seletor de classe é mais específico, os seletores de atributo e peseudoclasses tem o mesmo peso de especificidade.

Herança

Alguns valores e propriedades definidas para elementos pais são herdadas pelos seus filhos: color e font-family são herdados pelos elementos filhos, caso não tenha sido definidos diretamente.
