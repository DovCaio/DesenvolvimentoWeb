Box Model

Tudo em CSS está dentro de um quadro.

Caixas block inline

Temos dois tipos de caixas: caixas block e inline, isso influencia em como elas irão interagir com o fluxo da página.

Caixas block tem o seguinte comportamento:

  • A caixa irá quebrar em uma nova linha
  • A caixa irá se extender horizontamente (inline) para preencher todo o espaço disponível no container.
  • As propriedades width e height serão respeitadas.
  • padding, mergin e border farão com que outros elementos sejam empurrados para fora da caixa.

Se a caixa for do tipo inline, ela seguira os seguintes comportamentos

  • Ela não quebrará uma nova linha.
  • As propriedades width e heigth não serão aplicadas
  • Padding vertical, margens e bordas serão aplicadas, mas não farão com que outras caixas sejam empurradas
  • Padding horizontal, margens e bordas serão aplicadas e farão com que outras caixas embutidas se afastem

O tipo de caixa é definida pela propriedade display

Flexbox

Serve para a criação de tarefas de layouts de maneira mais fácil

Para que os elementos fiquem com o layous desejado devemos alterar a propriedade do pai colocando o display como flex, assim todos os itens desse elemento vão estar alinhados horizontalmente.

Flexbox tem uma propriedade chamada flex-direction que especifica a direção do eixo principal, por padrão é row, que faz que com eles fiquem alinhados horizontalmente.

Pode acontecer dos elementos ultrapassarem a tela do navegador, para resolver isso temos o flex-wrap

Às vezes precisamos que alguns itens dentro do layou tenha um tamanho diferente, para isso usamos a propriedade flex, para os elementos que queremos que tenha o mesmo tamanho damos o mesmo valor, e quando queremos um maior damos um valor maior.

Temos uma propriedade chamada aling-items a qual define a forma com que os itens vão se organizar, e junto com ela temos o justify-content que controla onde os elementos ficam ou seja, como se organizam.

Podemos também alterar a ordem a qual os elementos aparecem com a propriedade order, quando menor o valor do elementos, antes ele vai aparecer, caso elementos tenha a mesma order quem aparecer primeiro no html é quem vai aparecer primeiro também.

Partes de uma box

Content box
É o conteúdo que é mostrado, pode ser alterado com width e height
Padding box
É a parte que fica em volta do contúdo, pode ter seu tamanho alterado com padding
Border box
Fica em volta do Paddgin box, pode ter seu tamanho alterado pelo border
Margin box
É a última camada, tem seu tamanho alterado por margin

Margin

É uma parte invisivel que serve para separar os elementos, distânciar uns dos outros, o margin normal é um shorthand, temos diferentes propriedades dessa:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Podemos atribuir valores negativos também.

Border

O border é desenhado entre o padding e a margin, ela serve para fazermos exatamente uma borda.

Para ajustar o tamanho:

  • border-top
  • border-right
  • border-bottom
  • border-left

Também temos largura, estilo e cor:

  • border-direção-width
  • border-direção-style
  • border-direção-color

Caso não venhamos a colocar a direção a configuração vai servir para todas.

Padding

Fica entre a border e o conteúdo do elemento, diferentemente da margin não pode receber valores negativos, o background do elemento sera aplicado no padding também, o elemento fica atras do padding é usado para afastar o conteúdo da borda.

Temos também:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Inline block e o modelo de caixas

Todas as propriedades de caixa são aplicáveis a elementos inline block, porém vai ser colocado em cima dos outros elementos.

Ainda assim, podemos usar display inline-block assim o elemento não ficará em cima de nenhum também o width e o height são respeitados nesse display, e as propriedades de caixa vão empurar os outros elementos.
