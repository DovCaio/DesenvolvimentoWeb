Estilizando tabelas

Spaçamento e Layout

Temos a propriedade table-layout, define o algoritmo de layout a ser usado para organizar as células, colunas e, em alguns casos, até mesmo a tabela em si. Temos os seguintes valores:

auto
Modo padrão, o proprio navegador calcula automaticamente a largura das colunas.
fixed (mais recomendado)
A largura das colunas é definida explicitamente pelo desenvolvedor.

A border-collapse define como as bordas das colunas vão se comporatar entre si, o valor recomendado é collapse.

Usamos também border e padding para separar melhor os iten
